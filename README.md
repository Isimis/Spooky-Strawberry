# Spooky Strawberry Django

Custom e-commerce platform for Spooky Strawberry.

## Main modules

- Homepage
- Product catalog
- Product detail
- Product variants
- Ready-made outfits
- Cart
- Checkout
- Customer accounts
- SEO blog
- Custom admin dashboard
- AI assistant for product management

## Tech stack

- Python
- Django
- SQLite locally
- PostgreSQL in production
- Django Templates
- CSS
- JavaScript
